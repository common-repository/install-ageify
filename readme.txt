== Description ==
 
This plugin is intented to be used so that the AGEify age verification solution can be easily integrated to your WordPress website. 

By integrating AGEify on the homepage of your website you can reliably verify the age of your visitors and ensure that they are above or below a certain age, 

according to the policies that you create on the AGEify backend.

If you are interested in learning more on what AGEify is and how it works you can visit our website https://age-ify.com/ and even request a live demo if you want.

If you want to onboard your website you contact us here https://age-ify.com/#contact or send an email to info@age-ify.com
 

== Installation ==
 
1. Log into WordPress Dashboard and click on Plugins tab
1. Select zipped file of the plugin and press install
1. Activate the plugin through the 'Plugins' menu in WordPress
1. After Activation, a tab call Ageify will appear on Admin menu options bar. Click on it
1. Add the provided tracking code you got from info@age-ify.com and press save.
1. Now you should be able to see Age verification popup on your page.

If you have any trouble installing the plugin,download the more detailed manual from https://age-ify.com/ageify-documentation.docx
